Create A Painting Program

  • Step 1: Login to Scratch and click create to get a new project.
  • Step 2: Delete the Scratch Cat Sprite and Add a pencil sprite with a second costume of a magic wand (hint: For the second costume you will need to be in the Costume Editor Tab)
  • Step 3: Add the special paint blocks to our project.
  • Step 4: Now we will need to add our color changing Sprites. Find the button sprite and edit the costume for the color you want. (I recommend you add one for each color in a rainbow.)
  • Step 5: Next we will need to create the action buttons for Draw, Erase, Increase Size, Decrease Size, and Erase All. (hint: To add an image to a button you will need to copy and paste one costume onto another)
  • Step 6: Set the size of all buttons to 50% and line them up in two neat rows at the bottom of the screen. (hint: Each button takes up 50 pixels) Optional: Add a rectangle with some color to the bottom of the backdrop sprite.
  • Step 7: Get the pencil sprite to follow the mouse around the screen, but only when the mouse is above the buttons. (hint: You may need to check the Sensing and operational blocks. )
  • Step 8: Next get the pencil to draw a black line on the screen when you hold down the mouse button. (hint: This is where our new blocks will come in. )
  • Step 9: Let's get our line color to change when we click one of the paint color blocks.Make sure that the color matches the one clicked. (hint: It will probably be helpful here to have one sprite communicate to another. )
  • Step 10: Now we need to create the ability to erase when we click on the button with the magic wand. We will also want to change our pencil to the magic wand costume. (hint: Think about erasing as drawing with white. ) (hint: It will still probably be helpful here to have one sprite communicate to another. )
  • Step 11: Now let's get our pen size to increase when we click the plus button and decrease when we click the minus button. (hint: It will still be helpful here to have one sprite communicate to another. )
  • Step 12: Finally, let's erase the canvas completely when the lightning bolt button is clicked. (hint: Our newly added buttons for the pen will be helpful here. )

Bugs To Look Out For

  • Canvas not erasing when we begin.
  • Incorrect costume shows for draw.
  • Pencil continues to follow mouse even when attempting to click on buttons.

Ways to Customize

  • Change your pencil costumes to match the color it is drawing with.
  • Add a button to create stamps.
  • Show the pen size as a variable.
  • Add another button that let's you enter the number for pen size.
